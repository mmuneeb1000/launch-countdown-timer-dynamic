## Countdown Timer

A simple countdown timer built with React and Vite. It displays days, hours, minutes, and seconds with smooth flip animations when values update.

## Features

- Live countdown timer
- Days, hours, minutes, seconds breakdown
- Flip animation on value change
- Responsive layout
- Built with modern React setup

## Tech Stack

- React (JavaScript)
- Vite
- CSS (custom styling)

## Project Setup

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
